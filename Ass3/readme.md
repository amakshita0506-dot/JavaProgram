
#  Book Management System

## Description

This is a simple Java program demonstrating Object-Oriented Programming concepts such as classes, objects, ArrayList, and custom exception handling.
The system allows users to store and manage book details with proper validation for genre and price.

---

## Files

* **Book.java** → Contains the Book class with attributes and validation logic
* **ArrayListOfBooks.java** → Contains the main method to execute the program
* **InvalidGenreException.java** → Custom exception for invalid genre
* **InvalidPriceException.java** → Custom exception for invalid price

---

## How to Run

1. Compile the files:


javac ArrayListOfBooks.java,Book.java ,InvalidGenreException.java ,InvalidPriceException.java


2. Run the program:


java ArrayListOfBooks
