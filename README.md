# Java Calculator

This is a menu-driven calculator program written in Java.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Modulo operation

## Files
- Main.java – handles user input and menu
- Calculator.java – contains arithmetic methods

## How to Run
1. Compile the program:
   javac Main.java Calculator.java
2. Run the program:
   java Main
