

# Students CSV CRUD (Java)

## Author

Akshita Mishra

---

## Description

This project performs **CRUD operations (Create, Read, Update, Delete)** on a `Students.csv` file using Java.

---

## Features

* Add new students
* Display all records
* Update marks and calculate percentage
* Delete a student record
* Handle file exceptions

---

## Files

* `Students.csv` → Stores student data
* `StudentsCSVCRUD.java` → Java program

---

## How to Run

```bash
javac StudentsCSVCRUD.java
java StudentsCSVCRUD
```
