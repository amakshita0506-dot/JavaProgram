#  Banking Application (Java)

##  Overview

This project is a simple **Banking Application** developed in Java. It demonstrates core concepts of **Object-Oriented Programming (OOP)** such as classes, objects, inheritance, and encapsulation.

The application allows users to perform basic banking operations like:

* Deposit money
* Withdraw money
* Check account balance

---


##  Features

* Create bank account with account number and balance
* Deposit amount into account
* Withdraw amount with balance check
* Display current balance
* Handles insufficient balance cases

---

##  Concepts Used

* Classes and Objects
* Inheritance
* Encapsulation
* Method Overriding
* Conditional Statements

---

##  Code Explanation

###  Account Class

* Stores account number and balance
* Provides methods:

  * `deposit()` → Adds money
  * `withdraw()` → Deducts money
  * `getBalance()` → Returns balance

---

##  How to Run

### Step 1: Compile

```bash
javac BankingApp.java
```

### Step 2: Run

```bash
java BankingApp
```
