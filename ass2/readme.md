# Vehicle Management System

## Description
This is a simple Java program demonstrating Object-Oriented Programming concepts such as classes, objects, and methods.

## Files
- Vehicle.java → Contains the Vehicle class
- MainVehicle.java → Contains the main method to execute the program

## How to Run
1. Compile the files:
   javac MainVehicle.java Vehicle.java

2. Run the program:
   java MainVehicle
